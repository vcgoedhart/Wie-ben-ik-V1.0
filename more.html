Ik ben 16 jaar oud. Mijn hobby's zijn programmeren, basketballen en gamen. Ik vind de opleiding AO leuk om te doen omdat je iets leert waar je echt interesse in hebt. Motivatie

Ik vind het persoonlijk best wel leuk om met codes te werken. Ik wou al een tijdje HTML & CSS leren, omdat web designing toch wel interessant is. Omdat ik HTML & CSS eigenlijk toch al wilde leren was ik er al gemotiveerd voor. Het is ook nog een stuk leuker om het te leren met vrienden. De Java ervaring helpt me toch wel best wel erg om programma's te lezen en te begrijpen. Het is best leuk om bekende codes te zien van JS wat je eigenlijk al eerder gezien heb in Java.

Ontwikkelpunten

Ik zou graag meer willen leren van HTML & CSS, JS en PHP. Ik zou ook graag C# en Python willen leren maar dat doe ik wel in mijn vrije tijd. Het liefst zou ik mezelf willen verder ontwikkelen in JS. Want JS is echt programmeren en het kan ook worden gebruikt voor meerdere projecten. Zoals Unity bijvoorbeeld. Het liefst los ik mijn problemen zelf op, maar een extra hand die me op weg helpt vind ik ook niet super erg.

Bijbaan

Ik heb niet veel bijbanen gehad. Ik heb bij de 'Gemeente Alblasserdam' stage gelopen. Daar heb ik geleerd om zelfstandig te werken. De mensen daar kwamen bijna nooit naar me toe om me opdrachten te geven. Ik moest het zelf doen, dat heb ik ook vervolgens gedaan. Ik heb eigenlijk geleerd om opdrachten zelf uit te voeren zonder commando's van mijn stage begeleider. Dat is wel iets wat ik zou kunnen gebruiken in de opleiding Applicatie Ontwikkelaar.
